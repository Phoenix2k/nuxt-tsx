# Installation

# Cloning the project

Open a terminal window and clone the repository:

$ git clone --depth=1 https://github.com/Phoenix2k/nuxt-tsx.git project-name

# Install dependencies

Go to the newly created folder and install the project dependencies:

$ cd project-name
$ npm install

This will automatically create an .env file in the project root where you can set your Nuxt environment variables.
