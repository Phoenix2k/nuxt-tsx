# Documentation

This build comes bundled with VuePress, which you can use to document your application.

# Configuration

.
└─ docs
   └─ .vuepress
      └─ config.js

See Configuration for more information on how to configure your VuePress build.

# Development

$ npm run docs:dev

This will create a local server with hot-reload and reflect the changes you make while modifying these files.

# Build for production

$ npm run docs:build

This will build all files for production and place them in docs/.vuepress/dist.

# Theme

└─ .vuepress

   └─ styles
      └─ palette.styl
